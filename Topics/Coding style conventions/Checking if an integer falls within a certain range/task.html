Checking if an integer falls within a certain range

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Despite being a basic boolean operation, the precedence of logical operators (`&&`, `||`) can sometimes be confusing. Here's an unconventional way of teaching their precedence, using print statements. Your task is to implement a simple program in Java. The program should take a single integer as input, if the integer is greater than 5 and less than 10, it should print 'Correct Range', otherwise print 'Out of range'. Additionally, make sure to follow the Java coding style conventions.
Sample Input:
7

Sample Output:
Correct Range


Sample Input:
11

Sample Output:
Out of range
